# Project 2 - Quotespiration

#### A web page created for readers who love notating quotes. The aim of this app is to provide a space for users to keep a digital track of their book quotes.   

### Technologies Used:
- JavaScript
- Monogodb
- Mongoose
- Express 
- Node.js
